# Repro Disseny · marketing scaffold

Este paquete crea una base rápida de secciones para una web comercial en Nuxt.

## Instalación (copiar/pegar)
- Copia `components/marketing/*` a tu repo.
- Copia `layouts/default.vue` y `pages/index.vue` si quieres el ejemplo.

## Ajuste rápido
- Cambia textos y enlaces en props por defecto dentro de cada componente.
- Sustituye iconos (`<Icon name="lucide:..."/>`) por los que uses.
